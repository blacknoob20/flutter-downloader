# Changelog

## [1.0.0] - 2026-01-10

### Features
- ✨ Interfaz principal para ingresar URL de YouTube
- ✨ Selección inteligente de formatos recomendados
- ✨ Descarga de videos con seguimiento de progreso
- ✨ Historial completo de descargas
- ✨ Pausa/Reanuda/Cancela descargas
- ✨ Almacenamiento local de historial
- ✨ Soporte para tema oscuro/claro
- ✨ Validación de URLs de YouTube
- ✨ Integración con yt-dlp

### Architecture
- State management con Provider
- MVVM pattern con ViewModels
- Servicio-based architecture
- Manejo robusto de errores
- Logging detallado

### Tested
- Modelos de datos
- Validación de URLs
- Selección de formatos
- Manejo de descargas
